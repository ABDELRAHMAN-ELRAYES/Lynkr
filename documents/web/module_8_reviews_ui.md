# Module 8: Ratings & Reviews - Web GUI Specification

## 1. Overview
Social proof must look authentic and readable.

## 2. Components

### Review Components
*   **Star Input (Interactive)**:
    *   Hovering over the 4th star lights up 1-4.
    *   **Animation**: On click, stars do a small "Pop" scale effect.
    *   **Sentiment Text**: "Great!" appears when 4 stars selected. "Excellent!" for 5. "Poor" for 1.
*   **Distribution Bar**:
    *   Resume-style visual bars for "5 stars", "4 stars", etc.
    *   Line color: Gold/Yellow.

### Review Card
*   **Quote Context**:
    *   Put the review text in a subtle "Quote" bubble or use a large stylistic quotation mark background icon.
*   **Verified Purchase**:
    *   Green checkmark badge "Verified Job".

---

## 3. Premium UI Touches
*   **Filter by Rating**:
    *   Clicking the "5 Star" bar in the distribution chart filters the review list below to show only 5-star reviews.
*   **Response**:
    *   Provider responses appear indented with a "Author Response" badge and a slightly darker background.
